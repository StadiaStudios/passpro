Forgot Code but have the downloaded account?

Complex process. Requires backup and previous password if applicable, as long as you have the file.

  1. Download your account file via the settings within PassPro.
  2. Open the .txt file with a text editor such as VS Code, Notepad++, or any other text editor.
  3. Your passwords and usernames will be listed within the file. You can change the password by editing the input value in "PIN Lock: Null (Displays Your Password)".
  4. Change the Pin Lock number to "Null".

If you have already lost your account and have no backups or archived copies, unfortunately the account cannot be recovered without meeting minimum requirements.
