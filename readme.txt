How to Add Passwords:
Go to Dashboard, Type in Website/App Name. Then Type in Username/Email and Password. Press Add. The Password will now display inside the "Passwords" Section.

How to Save Passwords:
To save passwords and transfer between devices. Open Settings within the PassPro app, Press Download Account, Then do whatever you want with the file and whenever your ready to load it again or on a different device press load account and choose the .txt file. This also saves account usernames.
